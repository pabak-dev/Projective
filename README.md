# Projective: A Team-Oriented Task Management Platform

## 1. Project Overview

### Project Title

**Projective** – A Team-Oriented Task Management Platform

### Objective

The project aims to solve inefficiencies in team collaboration, communication, and accountability within office environments. By offering centralized task tracking, gamified incentives, and real-time communication, the platform enhances productivity and motivation.

### Target Audience

- Office managers and team leaders looking to manage projects and allocate work efficiently
- Team members who need clear visibility into tasks, deadlines, and team communication
- Small to medium-sized businesses seeking an affordable and intelligent task management tool

### Core Features

- **Kanban Board View:** Manage tasks efficiently across different categories
- **AI-Powered Chat Assistant:** Assist users with task assignments, project updates, and FAQs
- **Deadline Calendar View:** View all tasks and deadlines within a calendar
- **Points and Leaderboards:** Encourage healthy competition among employees

---

## 2. Detailed Features

### 2.1. Project Management

- Managers can create new projects and assign employees and team members
- Projects support Kanban views for easy progress tracking

### 2.2. Task Management

- Customizable fields: deadlines, priority, and assignees
- Status flow: To Do, In Progress, Review, Complete

### 2.3. User Roles & Permissions

- **Manager:** Full access to project and task management
- **Team Member:** Access to assigned tasks, with status updates and commenting ability

### 2.4. Gamification & Rewards System

- Tasks carry point values based on complexity or priority
- Points are awarded upon completion of the task
- Leaderboard to recognize high-performing team members

### 2.5. Communication & Collaboration

- Real-time chat for project-wide discussion
- Task-specific comment threads for focused updates

---

## 3. Technology Stack

- **Backend:** Laravel – Clean syntax and robust structure
- **Frontend:** React with Vite – Fast, modular development with optimized builds
- **Database:** Firebase Firestore / MongoDB – Real-time scalable NoSQL databases ideal for live collaboration features
- **Rendering Method:** Client-Side Rendering (CSR)
